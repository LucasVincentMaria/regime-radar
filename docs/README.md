## Add the dashboard screenshot

The main README references `docs/screenshot.png` (currently commented out). To show it:

1. Run the dashboard (`py scripts/run.py`) and open http://localhost:8000.
2. Take a screenshot of the full dashboard.
3. Save it here as `screenshot.png`.
4. In the top-level [`README.md`](../README.md), uncomment the line:
   `<!-- ![Regime Radar dashboard](docs/screenshot.png) -->`

A screenshot is the single biggest thing that gets people to try a project — worth doing.
